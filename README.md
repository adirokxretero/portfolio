## About This Project

This is my personal portfolio website — designed and built entirely by me
using my own imagination and ideas.

**Inspiration:** Took visual and interaction inspiration from
[utopiatokyo.com](https://www.utopiatokyo.com/) — particularly the
scroll animations, editorial typography, and dark aesthetic.

**Author:** Designed and developed by **Adithya M** — sole maintainer of this repository.

---

## Features

- ⚡ Glitch scramble intro screen
- 🖥️ Terminal-style logo (`root@adithya:~$ ./portfolio`)
- 🎯 Full-screen side navigation with number prefixes
- ✨ GSAP scroll animations & parallax reveal
- 🔴 Magnetic liquid-fill buttons
- 📊 Animated skill bars with categories
- 🃏 Flip-card project panels
- 🌗 Light / dark mode toggle (`</>` easter egg)
- 📍 Scroll progress indicator
- 📱 Fully mobile responsive
